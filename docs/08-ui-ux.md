## 초상화 활용 (920종 × 20감정)

| 상황 | 감정 |
|------|------|
| 영입 시 | 기쁨 (Joy) |
| 진화 시 | 놀람 → 기쁨 |
| HP 위험 | 울음 (Crying) |
| 보스 등장 | 분노 (Angry) |
| 사이클 클리어 | 감동 (Inspired) |
| 도감 등록 | 기쁨 |
