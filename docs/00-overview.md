# 00. 게임 개요

> **포켓몬 서바이버즈 (Poké Survivor)**
> 포켓몬 뱀서라이크 팬게임 | Phaser 3 | 모바일 웹 (세로)

## 한 줄 요약

**"5분마다 새 포켓몬을 골라 파티를 짓고, 보스를 잡고, 군단을 키워 끝없이 강해져라."**

## 핵심 정체성

| 항목 | 내용 |
|------|------|
| 장르 | 뱀서라이크 (Vampire Survivors / Brotato 류) |
| 플랫폼 | 모바일 웹 (Phaser 3), 세로(portrait) 기준 |
| 조작 | 한손 — 가상 조이스틱으로 이동만, 공격은 자동 |
| 시점 | 탑다운 2D, PMD 픽셀아트 스타일 |
| 한 사이클 | **5분** (보스 격파까지) |
| 플레이 구조 | 5분 사이클 × 무한 반복 |
| 타겟 | 포켓몬 팬 + 캐주얼 뱀서라이크 유저 |

## 뱀서라이크와의 차별점

| Vampire Survivors | Poké Survivor |
|---|---|
| 무기를 주워서 장착 | **동료 포켓몬**이 무기 역할 |
| 무기 레벨업 → 강화 | 동료 포켓몬 **진화** → 공격패턴 변화 |
| 무기 + 패시브 = 합성 무기 | 포켓몬 + 아이템 = **진화/폼체인지** |
| 캐릭터 고유 무기 1개 | **에이스 포켓몬** (매 사이클 선택) |
| 한 판 끝나면 처음부터 | **군단 누적** — 이전 파티가 영구 자산 |
| 20~30분 단판 | **5분 × 무한** — 끝이 없는 성장 |
