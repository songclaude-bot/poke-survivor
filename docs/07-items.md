# 07. 아이템 시스템

## 카테고리

### 지닌 물건 (Held Item) — 패시브

에이스에 장착. 최대 **4개**. 사이클 간 이월.

| 아이템 | 효과 |
|--------|------|
| 기합의띠 | HP 30% 이하 → DEF +50% |
| 구애스카프 | ATK +25%, 기술 변경 불가 |
| 달인의띠 | 상성 효과 ×1.5 |
| 조개껍데기방울 | 기술 히트 시 HP 회복 |
| 왕의징표석 | 공격 시 10% 풀죽음 |
| 넓은렌즈 | 크리티컬 +15% |
| 먹다남은음식 | 5초마다 HP 1% 회복 |
| 기합의머리띠 | 치명타 1회 생존 (HP 1) |
| 학습장치 | 경험치 +30% |
| 부적금화 | 골드 +25% |

### 나무열매 (Berry) — 소비

즉시 사용. 적/상자에서 드랍.

| 나무열매 | 효과 |
|----------|------|
| 오랭열매 | HP 30% 즉시 회복 |
| 리체열매 | ATK +20% (60초) |
| 자뭉열매 | 다음 1회 슈퍼이펙티브 무효 |
| 랑사열매 | 크리티컬 +30% (60초) |
| 수류열매 | 이동속도 +40% (30초) |
| 럼열매 | 상태이상 즉시 해제 |

### 진화석 — 동료 진화용

보스/미니보스 드랍, 아이템 상자.

```
물의돌, 불꽃의돌, 천둥의돌, 리프의돌,
달의돌, 태양의돌, 어둠의돌, 빛의돌,
메가스톤 (범용), 이상한사탕 (범용 진화)
```

## 아이템 드랍 규칙

| 소스 | 드랍 |
|------|------|
| 일반 적 | 경험치 보석 (100%), 나무열매 (1%) |
| 정예 적 | 경험치 보석, 나무열매 (5%), 지닌물건 (2%) |
| 아이템 상자 (1:30) | 지닌물건 or 진화석 (확정) |
| 미니보스 (2:30) | 지닌물건 (확정), 진화석 (30%) |
| 스테이지 보스 | 진화석 (확정), 메가스톤 (10%) |

## 아이템 이월 규칙

- **지닌 물건**: 사이클 간 이월 (에이스에 장착)
- **나무열매**: 소비 즉시 사라짐, 이월 없음
- **진화석**: 사용 안 했으면 이월 (인벤토리 최대 3개)
- **군단화 시**: 동료에게 사용된 진화석은 군단 파워에 반영

## 상태이상

### 디버프 (적에게 부여)

| 상태 | 효과 | 지속 |
|------|------|------|
| 화상 | 지속 데미지 | 3초 |
| 독 | 지속 데미지 (중첩) | 5초 |
| 마비 | 이동/공격 속도 -50% | 2초 |
| 빙결 | 행동 불가 | 1.5초 |
| 수면 | 행동 불가 (피격 시 해제) | 2초 |
| 혼란 | 이동 방향 랜덤 | 3초 |
| 풀죽음 | 공격 불가 | 2초 |
| 둔화 | 이동속도 -30% | 3초 |

### 버프 (플레이어에게)

| 상태 | 효과 | 소스 |
|------|------|------|
| 보호막 | 데미지 흡수 | 리플렉터, 빛의장막 |
| 분노 | ATK +30% | HP 낮을 때 자동 |
| 전기필드 | 전기 데미지 +30% | 군단 스킬 / 날씨 |
