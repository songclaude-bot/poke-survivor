# 01. 핵심 루프 — 5분 사이클 × 무한 반복

## 핵심 컨셉

> **5분 플레이 → 보스 격파 → 파티가 "군단"으로 압축 → 새 에이스 선택 → 다음 5분**
> 이것이 무한 반복. 매 사이클마다 군단이 하나씩 늘고, 적도 강해진다.

## 왜 5분인가

| 문제 (20분 단판) | 해결 (5분 사이클) |
|---|---|
| 모바일에서 20분은 긴 세션 | 5분 = 지하철 한 정거장 |
| 중간에 죽으면 처음부터 | 5분마다 체크포인트 (보스 격파 = 저장) |
| 한 판에 에이스 1마리 → 리소스 낭비 | 5분마다 새 에이스 → 더 많은 포켓몬 사용 |
| 빌드 완성 후 늘어짐 | 빌드 완성 직전에 보스 → 긴장감 유지 |
| "클리어" 후 할 게 없음 | 영원히 다음 사이클이 있음 |

## 전체 흐름도

```
사이클 1 (5분) — "작은숲"
┌─────────────────────────────────┐
│ 에이스: 피카츄                    │
│ 동료 영입: 꼬부기, 파이리, 이브이  │
│ 보스: 카이로스 격파!               │
└────────────┬────────────────────┘
             ▼  파티 → 군단①로 압축
┌─ 사이클 전환 화면 ──────────────┐
│  "군단 결성!"                     │
│  [피카츄 군단] — 전기/물/불/노말   │
│  새 에이스를 고르세요:             │
│  [리자몽]  [라프라스]  [루카리오]  │
└────────────┬────────────────────┘
             ▼
사이클 2 (5분) — "해변동굴"
┌─────────────────────────────────┐
│ 에이스: 리자몽                    │
│ 군단①: 피카츄 파티 (자동 전투)    │
│ 동료 영입: 또가스, 팬텀, 메타몽   │
│ 보스: 갸라도스 격파!               │
└────────────┬────────────────────┘
             ▼  파티 → 군단②로 압축
사이클 3, 4, 5... ∞
```

## 핵심 규칙

| 규칙 | 내용 |
|------|------|
| 사이클 시간 | **고정 5분** (변하지 않음) |
| 매 사이클 시작 | 새 에이스 1마리 선택 |
| 매 사이클 중 | 동료 최대 3마리 영입 |
| 매 사이클 끝 | 보스 격파 → 현재 파티(에이스+동료3) → 군단 1개로 변환 |
| 군단 | 이전 사이클의 파티. 축소 형태로 자동 전투. 계속 누적 |
| 실패 시 | 현재 사이클만 재도전 (군단은 유지) |
| 게임오버 | 없음. 원할 때 그만둠. 자동 저장 |

## 5분 사이클 타임라인

```
시간   이벤트
─────┬──────────────────────────────────
0:00 │ 시작. 에이스 1마리 + 군단 출격
0:15 │ 첫 레벨업 → 동료 1마리 영입
0:30 │ 적 밀도 증가
1:00 │ ★ 엘리트 등장 (정예 3마리)
1:15 │ 레벨업 → 동료 2마리째 or 강화
1:30 │ 아이템 상자 드랍
2:00 │ 적 강화 (티어 업), 날씨 변화
2:30 │ ★ 미니보스 (준전설급)
2:45 │ 레벨업 → 동료 3마리째 or 진화
3:00 │ "경고!" — 대량 스폰
3:30 │ 진화석 드랍 (마지막 강화 기회)
4:00 │ ★★ 스테이지 보스 등장
4:30 │ 보스 페이즈 2
5:00 │ 타임리밋. 미격파 시 사이클 실패
─────┴──────────────────────────────────
```

## 파티 구성 (에이스 + 동료 3)

5분에 맞춰 동료를 **3마리**로 축소:
- 매 선택이 무겁다 (전체의 33%)
- 타입 조합 전략이 간결 (트라이앵글)
- 군단이 쌓이면서 총 유닛 수는 자연히 늘어남

## 레벨업 선택지 (3개 중 택 1)

```
슬롯이 비어있을 때:
[A. 새 동료 영입] [B. 기존 동료 강화] [C. 에이스 강화]

슬롯 3개 다 찼으면:
[B. 동료 강화] [C. 에이스 강화] [D. 군단 버프]
```

**군단 버프** — 사이클 후반에만 등장:
- 군단① ATK +20%
- 군단② Speed +30%
- 전 군단 HP 회복 50%

## 사이클 전환 — 3단계

```
STEP 1: 보상 수령
  처치 수, 시간, 신규 도감, 스타더스트

STEP 2: 군단 결성
  현재 파티 → 군단화, 군단 스킬 부여

STEP 3: 다음 사이클 준비
  스테이지 확인, 새 에이스 선택(3종 중 택1), 아이템 장착
```

## 죽음 & 재도전

```
보스 타임오버 또는 에이스 HP 0
  ├─ [재도전] — 같은 사이클, 군단 유지, 에이스/동료 리셋
  ├─ [군단 강화] — 스타더스트로 군단 일시 버프
  └─ [철수] — 메인 메뉴 (진행 저장, 나중에 이어하기)
```
