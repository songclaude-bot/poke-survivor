# 03. 기술/무기 시스템

## 설계 철학

> 기술 = 동료 포켓몬의 자동 공격 패턴.
> 오토체스 536개 기술을 뱀서라이크에 맞게 재분류.

## 동료 기술 성장 (Lv.1 → Lv.3 + 진화)

5분 사이클에 맞춰 **최대 Lv.3 + 진화** (간소화).

### 예시: 꼬부기 (프로젝타일)

| Lv | 기술 | 효과 |
|----|------|------|
| 1 | 물대포 | 단일 투사체, 1초 간격 |
| 2 | 물의파동 | 크기 ×1.5, 둔화 부여, 2연발 |
| 3 | 하이드로펌프 | 관통, 데미지 ×2, 3연발 부채꼴 |
| +진화석 | **오리진펄스** | 화면 관통 3줄기 + 폭발 |

### 예시: 또가스 (오비탈)

| Lv | 기술 | 효과 |
|----|------|------|
| 1 | 독가스 | 1마리 회전, 접촉 데미지 |
| 2 | 스모그 | 2마리 회전, 독 상태이상 |
| 3 | 헤도로웨이브 | 3마리 회전, 독 강화, 속도 2배 |
| +진화석 | **독폭탄** | 5마리 회전 + 폭발 잔류장 |

|-------------|---------|------------|
| 꼬부기 | + 신비의물방울 | **시작의바다** |
| 피카츄 | + 전기구슬 | **1000만볼트** |
| 또가스 | + 독바늘 | **독안개** |
| 이상해씨 | + 기적의씨앗 | **꽃보라** |
| 팬텀 | + 주술의부적 | **어둠의구멍** |
| 루카리오 | + 메가스톤 | **메가파동** |

## PP/쿨다운 시스템

```
일반 공격 (Speed 기반 쿨다운)
  ├─ 적 히트 → PP +5
  ├─ 피격 → PP +10
  └─ 시간 경과 → PP +1/초

PP ≥ MaxPP → 기술 발동!
  └─ PP 리셋, 재충전 시작
```

- Speed 낮음 (≤40): 느린 공격, 높은 단일 데미지
- Speed 높음 (≥60): 빠른 공격, 낮은 단일 데미지
- MaxPP 낮음 (50~70): 기술 자주 시전 (스킬 위주)
- MaxPP 높음 (100~120): 기술 가끔 but 강력 (궁극기형)

## 타입 상성

### 배율 (완화)

| 관계 | 배율 |
|------|------|
| 효과 좋음 | ×1.3 |
| 효과 없음 | ×0.8 |
| 면역 | ×0.5 (완전 면역 아님) |

### 타입 시너지

| 같은 타입 동료 수 | 효과 (불 예시) |
|------------------|---------------|
| 2마리 | 화상 확률 +15% |
| 3마리 | 불 데미지 +20%, 드래곤 추가 +30% |

레인보우 전략 (다양한 타입 혼합):

| 서로 다른 타입 수 | 보너스 |
|------------------|--------|
| 3 타입 | 경험치 +10% |
| 4 타입 (에이스 포함) | 아이템 드랍률 +15%, 모든 스탯 +10% |
