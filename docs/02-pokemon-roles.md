# 02. 포켓몬 역할 분배 — 1,000마리 활용

## 설계 철학

> 매 사이클 새 에이스를 고르고, 동료 3마리를 영입하고, 적 수백 마리를 상대한다.
> 사이클이 쌓일수록 **자연스럽게 수백 종이 플레이에 관여**하게 된다.

## 역할 분배 맵

| 역할 | 수량 | 설명 |
|------|------|------|
| **에이스** (플레이어) | ~60종 | 매 사이클 선택하는 주인공 |
| **동료** (무기) | ~200종 (진화 포함 ~400) | 레벨업 시 영입, 자동 공격 |
| **적** (몹) | ~600종 | 웨이브로 밀려오는 야생 포켓몬 |
| **보스** | ~60종 | 사이클 끝 보스 + 미니보스 |
| **도감 전용** | ~40종 | 특수 조건 등장 희귀 포켓몬 |

> 역할 중복 가능 — 피카츄가 에이스/동료/적 모두 가능

## 에이스 포켓몬 — 60종

에이스 = 뱀서의 "캐릭터 선택". 매 사이클마다 다른 에이스 사용.

### 초기 해금 (6종)

| 에이스 | 타입 | 스타일 | 에이스 기술 | 패시브 |
|--------|------|--------|------------|--------|
| 피카츄 | 전기 | 근접 속공 | 볼트태클 (돌진+광역) | 정전기: 피격 시 10% 마비 |
| 이상해씨 | 풀 | 원거리 지속 | 솔라빔 (관통) | 엽록소: 쾌청 시 ATK+25% |
| 파이리 | 불 | 근접 폭딜 | 화염방사 (부채꼴) | 맹화: HP 30%↓→ATK+50% |
| 꼬부기 | 물 | 원거리 탱커 | 하이드로펌프 (대형 관통) | 급류: HP 30%↓→물 데미지+50% |
| 이브이 | 노말 | 런 중 분기진화 | (진화에 따라 변화) | 적응력: 타입 일치 보너스 2배 |
| 뮤 | 에스퍼 | 기술 랜덤 | 매 사이클 랜덤 기술 | 싱크로: 상태이상 반사 |

### 이브이 — 런 중 진화 분기

```
이브이 (노말)
 ├─ 물의돌 → 샤미드 (물)
 ├─ 불꽃의돌 → 부스터 (불)
 ├─ 천둥의돌 → 쥬피썬더 (전기)
 ├─ 에스퍼 동료 보유 → 에브이 (에스퍼)
 ├─ 악 타입 스테이지 → 블래키 (악)
 ├─ 풀 스테이지 → 리피아 (풀)
 ├─ 얼음 스테이지 → 글레이시아 (얼음)
 └─ 페어리 동료 보유 → 님피아 (페어리)
```

### 에이스 선택 규칙

- 매 사이클 3종 중 택 1 (스테이지 타입 가중치)
- 이미 군단 에이스로 사용한 포켓몬은 선택지에 안 나옴
- 도감 진행에 따라 선택지 풀 확장
- 해금 조건: 도감 등록 수, 스테이지 클리어, 보스 격파 등

## 동료 포켓몬 (무기) — 6가지 아키타입

| 아키타입 | 설명 | 뱀서 대응 | 대표 예시 |
|----------|------|-----------|----------|
| **오비탈** | 에이스 주위 회전, 접촉 데미지 | 성경/마늘 | 또가스, 레어코일, 루나톤 |
| **프로젝타일** | 가까운 적에게 투사체 발사 | 매직 완드 | 꼬부기, 피카츄, 팬텀 |
| **에어리어** | 주기적 광역 데미지 | 마늘/성수 | 잠만보(지진), 라프라스(파도타기) |
| **서몬** | 독립 이동, 적 추적 공격 | 고양이/비둘기 | 메타몽, 로토무 |
| **배리어** | 방어막 + 반사 데미지 | 룬트레이서 | Mr.마임, 라플레시아 |
| **노바** | 전방위 폭발 (고위력·긴 쿨) | 번개반지 | 마그마(분화), 레지기가스 |

## 적 티어 시스템

| 티어 | 출현 시점 | 행동 | 예시 |
|------|-----------|------|------|
| 일반 | 0분~ | 직진 추적 | 캐터피, 꼬렛, 주뱃 |
| 속도 | 0:30~ | 빠른 이동 | 디그다, 나옹 |
| 탱커 | 1분~ | 느림, 고HP | 꼬마돌, 코코리 |
| 원거리 | 1:30~ | 멈춰서 투사체 | 메노크라게, 슬리프 |
| 정예 | 2분~ | 소보스급 | 리자드, 후딘 |
| 군체 | 3분~ | 대량, 저HP | 미니리트 떼, 주뱃 떼 |
| 돌진 | 3:30~ | 화면 밖 돌진 | 켄타로스, 보만다 |

### 적 출현 로직

```
스테이지 타입에 따라:
├─ 1차 타입 적 (60%): 스테이지 메인 타입
├─ 2차 타입 적 (25%): 관련 타입
├─ 범용 적   (10%): 노말
└─ 역상성 적  (5%): 보너스 보상 포켓몬
```

## 보스 등급

| 등급 | 출현 | HP 배율 | 예시 |
|------|------|---------|------|
| 미니보스 | 2:30 | ×50 | 앱솔, 메타그로스 |
| 스테이지보스 | 4:00 | ×200 | 뮤츠, 레쿠쟈 |
| 세대보스 | 사이클 12 | ×500 | 아르세우스 (전 타입 변환) |

## 도감 & 수집

모든 포켓몬을 만남/사용/격파 시 도감 등록.

### 도감 마일스톤

| 등록 수 | 보상 |
|---------|------|
| 50종 | 새 스테이지 해금 |
| 100종 | 에이스 선택지 +1 |
| 200종 | 나무열매 레시피 |
| 500종 | 전설 에이스 해금 (뮤) |
| 1000종 | 아르세우스 에이스 해금 |
