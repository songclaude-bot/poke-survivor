# 06. 스테이지 & 맵 구성

## 구조

한 세대(12사이클)에 12개 스테이지를 순차 진행.
각 스테이지 = **타일셋 + 타입 테마 + 날씨 + 적 + 보스 + BGM + 기믹**

## 스테이지 상세 예시

### 사이클 4: 화산

```
타일셋: MtBlaze + MagmaCavern
타입: 불 / 바위
날씨: DROUGHT (가뭄)

적:
├─ 60%: 불 — 파이리, 마그마그, 마그비, 불카모스
├─ 25%: 바위 — 꼬마돌, 코코리, 이시쯔부테
├─ 10%: 노말 — 꼬렛, 이어롤
└─  5%: 역상성(물) — 꼬부기 (보너스 보상)

날씨 효과: 불 적 ATK+20%, 불 기술 강화, 물 동료 기회
기믹: 랜덤 위치에 용암 웅덩이 (지속 데미지 지형)

보스: 앤테이
├─ HP ×200
├─ 기술1: 성스러운불꽃 (십자 범위)
├─ 기술2: 화염방사 (직선)
└─ Phase 2: 오버히트 (전방위 폭발)
```

## 5분 안에서의 배경 전환

```
0:00~2:00  밝은 톤 (기본 타일셋)
2:00~4:00  어두운 변형 (위기감 연출)
4:00~5:00  보스 전용 배경 (다크 버전 + 색조 변경)
```

## 날씨 시스템

사이클 시작 시 결정, 중간 변화 1회:

```
0:00 — 기본 날씨 (스테이지에 따라)
2:30 — 날씨 변화 (강화 또는 보스 연출)
```

### 날씨별 효과

| 날씨 | 비주얼 | 효과 |
|------|--------|------|
| RAIN | 비 파티클 | 물 +25%, 불 -25% |
| DROUGHT | 열기 일렁임 | 불 +25%, 물 -25% |
| SANDSTORM | 모래 파티클 | 전원 지속피해 (바위/땅/강철 면역) |
| SNOW | 눈 파티클 | 얼음 DEF+25%, 전체 이동속도 -15% |
| STORM | 번개+비 | 전기 범위 +50% |
| MISTY | 안개 | 드래곤 -50%, 상태이상 면역 |
| NIGHT | 어두운 화면 | 악 +30%, 시야 축소 |
| WINDY | 바람 파티클 | 투사체 속도/방향 영향, 비행 +30% |
| SMOG | 독안개 | 전원 독 피해, 독 타입 면역 |
| BLOODMOON | 붉은 달 | 적 ATK+30%, 드랍률 2배 |

## 맵 방식

**무한 스크롤**: 타일셋 반복 타일링, 플레이어 중심 적 스폰.

```
맵 위 오브젝트:
· 경험치 보석 (적 처치 드랍)
· 아이템 상자 (1:30에 1회)
· 나무열매 (HP 회복, 랜덤)
· 진화석 (보스/미니보스 드랍)
```
