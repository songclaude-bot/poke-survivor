# 04. 군단 시스템 — 재귀 성장의 핵심

## 설계 원칙

> 군단은 "과거의 나"가 남긴 유산이다.
> **시각적으로 화려하되, 연산은 가볍게.** 이것이 무한 플레이의 핵심.

## 군단이란?

사이클 보스를 격파하면 현재 파티(에이스+동료3)가 **군단**으로 변환된다.
군단은 영구히 보존되며, 이후 모든 사이클에서 자동으로 전투에 참여한다.

```
사이클 1 파티 → 군단①
사이클 2 파티 → 군단②
사이클 3 파티 → 군단③
...
사이클 N 파티 → 군단⑩⑩⑩
```

## 화면 표현 — 3단계 LOD

### 상세 모드 (최근 2개 군단)

```
군단 멤버 4마리가 풀 스프라이트로 전투
에이스를 느슨하게 따라다니며 독립 공격

[동료] [에이스] [동료]
       [동료]
```

### 심플 모드 (3~8번째 군단)

```
에이스 스프라이트 1개 + 타입 색상 오라
주기적 광역 공격 발사

🟡 ← 피카츄 군단 (전기 오라)
🔴 ← 리자몽 군단 (불 오라)
```

### 추상 모드 (9번째 이상)

```
빛나는 점(파티클 1개)
에이스 주변에 성좌처럼 배치
자동 광역 펄스

  · ★ ·
★   [에이스]  ★
  ·  ★  ·
```

### 사이클 16+: 빛의 고리

모든 추상 군단이 에이스 주변의 **광채 링** 하나로 통합.
셰이더 이펙트 1개로 렌더링.

## 군단 전투 행동

| LOD | 군단 수 | 전투 방식 | 렌더 비용 |
|-----|---------|-----------|-----------|
| 상세 | 최근 2개 | 개별 4마리 독립 이동+공격 | 8 스프라이트 |
| 심플 | 3~8번째 | 0.5초마다 범위 AOE 데미지 | 6 스프라이트 |
| 추상 | 9번째~ | 1초마다 글로벌 DPS 적용 | 파티클만 |

**렌더링 상한: 항상 ~150 오브젝트 이하**

## 군단 스킬 (영구 패시브)

각 군단 결성 시 **타입 기반 영구 패시브** 부여:

| 군단 에이스 타입 | 군단 스킬 | 효과 |
|-----------------|-----------|------|
| 전기 | 전기 필드 | 전기 데미지 +10%, 피격 시 10% 마비 |
| 불 | 화염의 의지 | 불 데미지 +10%, HP 낮을 때 ATK↑ |
| 물 | 해류 | 물 데미지 +10%, 이동속도 +5% |
| 풀 | 광합성 | 풀 데미지 +10%, HP 0.5%/초 회복 |
| 격투 | 투지 | 격투 데미지 +10%, 크리티컬 +5% |
| 에스퍼 | 초감각 | 에스퍼 데미지 +10%, 경험치 +5% |
| 드래곤 | 용의 위엄 | 드래곤 데미지 +10%, 보스 데미지 +10% |
| 고스트 | 그림자의 저주 | 고스트 데미지 +10%, 적 Speed -5% |
| 얼음 | 절대영도 | 얼음 데미지 +10%, 빙결 확률 +5% |
| 독 | 맹독 | 독 데미지 +10%, 독 지속 데미지 +15% |
| 땅 | 대지의 힘 | 땅 데미지 +10%, DEF +5% |
| 바위 | 견고함 | 바위 데미지 +10%, HP +5% |
| 강철 | 강철 방어 | 강철 데미지 +10%, 피해 감소 3% |
| 비행 | 순풍 | 비행 데미지 +10%, 회피 +3% |
| 페어리 | 요정의 축복 | 페어리 데미지 +10%, 상태이상 면역 +10% |
| 악 | 어둠의 맹세 | 악 데미지 +10%, 치명타 데미지 +10% |
| 벌레 | 군집 본능 | 벌레 데미지 +10%, 동료 공속 +3% |
| 노말 | 적응력 | 전 타입 데미지 +3% |

> 사이클이 쌓일수록 군단 스킬도 누적 → 자연스러운 무한 성장

## 군단 DPS 계산

```
단일 군단 DPS = 결성시_파워 × (1 + 0.05 × 경과_사이클)

총 군단 DPS = Σ(각 군단 DPS)

예시 (사이클 10):
  군단① (사이클1 결성): 80 × 1.45 = 116
  군단② (사이클2 결성): 90 × 1.40 = 126
  ...
  군단⑨ (사이클9 결성): 150 × 1.05 = 158
  총 군단 DPS ≈ 1,200

적 전투력: BASE × 1.4^10 ≈ 2,300
현재 파티 DPS: ~500
총 플레이어: 1,700 vs 적 2,300 → 빡빡하지만 가능
```
